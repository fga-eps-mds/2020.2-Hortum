    3 Detalhamento e Justificativas

    3.1 Nome da Aplicação

       Para o nome da aplicação foi traduzido a palavra "horta" para latim "hortum holerum" e escolhido a primeira palavra "hortum". A ideia de buscar um nome referente a horta vem do perfil dos usúarios da plataforma, em especial os pequenos produtores, que possuem sua renda baseada em sua propriedade.

    3.2 Fontes

       As fontes utilizadas na aplicação são:

    • Comfortaa: (version 3.001, part of the Google Font Improvements Project), para títulos.
    • Roboto: (Grátis, Google Android Design), para corpo do texto.

       As fontes são de fácil visualização e de uso gratuito.

    Uso das fontes

    Incluindo no HTML

       Usa-se o seguinte trecho de código:

    <link rel="stylesheet" href="//fonts.googleapis.com/css?family=Comfortaa|Roboto" rel="stylesheet"/>

    Incluindo no CSS

       Usa-se no CSS:

    font-family: 'Comfortaa';
font-family: 'Roboto', sans-serif;
